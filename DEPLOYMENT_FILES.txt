USAGE INSTRUCTIONS
========================================

1. Review all documentation starting with:
   docs/deployment/README.md

2. Configure environment variables:
   - Copy .env.production.example
   - Fill in all values
   - Validate: npm run validate:env

3. Set up GitHub secrets:
   - Follow docs/deployment/GITHUB_SECRETS_SETUP.md
   - Add all required secrets

4. First deployment:
   - Use docs/deployment/CHECKLIST.md
   - Test on staging first
   - Monitor closely

5. For detailed procedures:
   - See docs/deployment/DEPLOYMENT.md
